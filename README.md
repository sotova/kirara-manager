# まんがタイムきらら作品管理サイト（Kirara Works Manager）

このプロジェクトは、「まんがタイムきらら」シリーズのコミックス作品やアニメ化作品を、プレミアムなビジュアルで楽しく管理できるサーバー不要のWebアプリケーションです。

読んだ巻数やアニメの視聴状況、マイ評価、感想・メモなどをブラウザ（LocalStorage）に完全無料で安全に保存できます。また、ブラウザ上で**すべてのコミックス情報を手動で「追加・編集・削除」できるマニュアルCRUD機能**が標準搭載されており、外部クローラーに依存せずに本棚を完全に管理可能です（補助用として、最新の `dokidokivisual.com` からデータを自動取得するためのNode.jsベースのローカルスクレイピングツールも付属しています）。

---

## 🎨 プレミアムなデザインと主な機能

1. **極上のビジュアル体験（Midnight Orchid & Sweet Kirara テーマ）**
   - きららテイストである「ビビッドピンク」と「深みのあるバイオレット」をブレンドした、プレミアム感溢れるダークモードUI。
   - ガラスのような質感を生むグラスモルフィズム、スムーズなホバーアニメーション、微小なマイクロインタラクション。
   - 絵文字を排除し、美しく洗練されたベクターSVGの **[Lucide Icons](https://lucide.dev/)** を完全採用。

2. **変幻自在の表示モード切り替え**
   - **タイトル別（Grouped View）**: 同じ作品（例: 『ぼっち・ざ・ろっく！』）を1つのカードにまとめてスッキリ表示。クリックすると全巻リストが現れ、それぞれの巻ごとに記録を行えます。
   - **1巻ずつ表示（Individual View）**: すべての書籍の各巻をフラットにグリッド表示し、コレクションのように一括表示できます。

3. **きめ細やかなステータス管理**
   - 🟢 **読んだ** (Read) / 視聴完了
   - 🟡 **読みたい** (Want) / 見たい
   - 🔴 **続かなかった** (Dropped) / 中断
   - ⚪ **未読** (Unread) / 未視聴

4. **アニメ作品の管理 & マンガとのスマートリンク**
   - アニメ化された作品を手動で簡単登録（タイトル、放送時期、全話数、視聴状況、評価、感想）。
   - **原作マンガとの連動**: 登録時に関連するマンガ作品を選ぶと、マンガの詳細画面からも自動的にアニメ化情報や自分の視聴ステータス、評価が連動して表示されます。

5. **バックアップ・JSONインポート・エクスポート**
   - 記録したすべての読書・視聴データをワンクリックで JSON ファイルとしてエクスポート。
   - PCの買い替えや他ブラウザへの移行時にも安全にデータを復元できます。

## 🚀 使い方（今すぐ使う方法）

本アプリはサーバーを必要としない静的Webアプリ（SPA）です。

1. 本ディレクトリ `kirara-manager` の中にある **`index.html`** を、お使いのブラウザ（Chrome, Edge, Safari等）でダブルクリックして開くだけで即座に起動します！
2. **購入直後でもすぐにお試しいただけるよう、あらかじめ人気きらら作品の書籍・巻数データ（計13冊分）を収録したデータベースファイル (`kirara_data.json`) を同梱しています。** スクレイピング作業を行わなくても、すぐにアプリの全機能をご利用いただけます。

## 🕷️ 最新データの取得：ローカルスクレイピング手順 (`scraper.js`)

`dokidokivisual.com`（芳文社公式サイト）から最新のコミックス情報を完全に無料かつ安全に取得してデータベースをアップデートするための手順です。

### 1. 事前準備
パソコンに **Node.js** がインストールされている必要があります。
インストールされていない場合は、[Node.js 公式サイト](https://nodejs.org/) から推奨版（LTS）をダウンロードしインストールしてください。

### 2. ライブラリのインストール
コマンドプロンプト（Windows）やターミナル（Mac）を開き、本プロジェクトのフォルダに移動して以下を実行します。

```bash
# 本フォルダに移動（※パスはご自身の環境に合わせて調整してください）
cd C:\Users\rn144\.gemini\antigravity-ide\scratch\kirara-manager

# 必要なライブラリをインストール
npm install axios cheerio iconv-lite
```
*※ `iconv-lite` は、芳文社公式サイトの Shift_JIS エンコーディングによる日本語文字化けを100%防ぐために使用します。*

### 3. スクレイパーの実行
以下のコマンドを実行します。

```bash
node scraper.js
```

**【スクリプトの動作について】**
- 一覧ページから作品詳細へのリンク（cid）を自動探索し、各書籍の **「タイトル、巻数、著者、レーベル、発売日、あらすじ、表紙カバー画像」** を解析・取得します。
- **サーバーに負荷をかけないマナー設計**: 1ページ巡回するごとに 1.5秒 のウェイト（スリープ）が入るため、アクセス拒否やサーバーエラーを起こさずに安全に動作します。
- 実行が完了すると、抽出した最新データが既存のデータベースと自動マージされ、`kirara_data.json` が上書き更新されます。

## 🛠️ 技術スタック

- **フロントエンド骨組み**: HTML5 (セマンティックタグ)
- **スタイリング**: バニラCSS (CSS Variables, Backdrop-filter, Grid/Flexbox Layout)
- **ロジック & 状態管理**: バニラJavaScript (ES6 Modules, LocalStorage)
- **グラフィック**: Lucide Icons (CDN連携)
- **スクレイピング**: Node.js + Axios (HTTPクライアント) + Cheerio (HTMLパーサ)
